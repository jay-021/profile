## 🎯 Learning Outcomes

Upon completing this programme, graduates are able to create, implement, and maintain secure information systems, prevent cyber attacks, and manage data security risks. Key competencies include secure network and IT infrastructure design, vulnerability analysis, firewalls, IDS/IPS systems, manual and automated testing, reverse engineering, database design, and IT security consulting.
