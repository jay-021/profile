## 📚 What It Demonstrates

AWS Cloud Quest is a **role-playing game** where you solve real cloud challenges — not just theory. Completing it proves:

- Deploying and configuring actual AWS services (EC2, S3, RDS, VPC, IAM, Lambda)
- Applying cloud best practices in guided scenarios
- Hands-on familiarity with the AWS Management Console
- Problem-solving approach to cloud architecture decisions

> [!tip] Differentiator 🇩🇪
> Most candidates have the AWS CCP cert. Cloud Quest proves *hands-on* skills — a stronger signal for recruiters and hiring managers at German tech companies.

> [!note] AWS Cert Stack
> AWS Knowledge: Cloud Essentials (Jul 2023) → **AWS Cloud Quest: Cloud Practitioner (Jun 2024)** → AWS Certified Cloud Practitioner (Jul 2023, expires Jul 2026) ✅
