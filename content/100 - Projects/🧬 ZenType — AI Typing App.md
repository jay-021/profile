# 🧬 ZenType — AI Typing App

> [!success] Status: Live & Deployed
> A serverless, GDPR-compliant AI typing web application targeting the European market.

---

## 🎯 What It Does

ZenType dynamically generates customized typing content using **Google Gemini AI**, giving users a unique, intelligent typing practice experience. Built to EU data standards from day one.

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 15, Tailwind CSS |
| Backend | Firebase (serverless) |
| AI | Google Gemini AI (dynamic content generation) |
| Auth/RBAC | Firebase Auth + Admin Controls |
| Compliance | GDPR — EU-based data storage |
| DevOps | CI/CD via GitHub Actions |

## 🏗️ Architecture Highlights

> [!info] Key Engineering Decisions
> - **Serverless-first**: Zero backend servers to maintain — Firebase Functions handle all logic
> - **GDPR by design**: EU-region data storage, automated data export/deletion pipelines
> - **RBAC admin controls**: Role-based access for content moderation
> - **AI content generation**: Gemini API generates typing passages on-demand per user profile
