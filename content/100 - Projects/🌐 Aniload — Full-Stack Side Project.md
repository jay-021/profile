## 🏗️ What It Shows

> [!example] Full-stack ownership — backend → web → mobile
> Built across university Semester 3. Three separate repos = frontend, backend, and mobile app sharing the same database.

- **API design**: REST backend serving both web and mobile clients
- **React Native**: Cross-platform mobile (same codebase for iOS + Android)
- **Shared database**: Single backend powers web and app
- **Self-initiated**: Not a course project — built out of interest
